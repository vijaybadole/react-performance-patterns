# Lazy Loading

Route-based and component-based code splitting with React.lazy and Suspense.

## Concepts

- **React.lazy()** - Dynamically import components
- **Suspense** - Manages loading states while lazy components load
- **Code Splitting** - Break bundle into smaller chunks loaded on demand

## Benefits

- Reduces initial bundle size
- Improves Time to Interactive (TTI)
- Load code only when needed

See `LazyRoute.jsx` for a practical implementation using navigation routes.
